电脑手机互传文件

不需要实物的数据线，电脑与手机间可轻松传文件；
对传输界面全面改版，让查看更清晰，体验更顺畅。
电脑端：点击主面板："我的设备" 即可
手机端：动态中找到 "传文件到我的电脑" 即可进行传输
